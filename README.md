# Clan-System
Erstelle ein Clan und Verwalte ihn
Commands:
- `/clan`: öffne das generelle Clan-Menü
- `/clan invite <Spieler>`: Lade ein Spieler in deinen Clan ein
- `/clan create <ClanTag> <ClanName>`: Erstelle einen Clan
